Phase 2 Deep Dive

6-8 Week Sprint Plan

Core Intelligence

  • Replanning Agent 核心邏輯

Notification & UI

  • Notification Agent 實作

  • Human-in-the-loop 流程

WEEK 7-8

Real-world & Stress Test

  • 真實 API 接入
  • 壓力測試與延遲優化
  • Fallback 機制驗證
  • End-to-end 整合測試

External Resources

Ready-to-Integrate APIs

MVP 所需的外部資料源均有成熟的商業 API 可直接整合，無需重複造輪子。

✈ Flight Status APIs
  • FlightAware AeroAPI 即時航班追蹤與 Webhook 推播 flightaware.com/commercial/aeroapi
  • Cirium 企業級全球航班狀態與歷史準點率數據 cirium.com
  • Aviationstack 輕量 REST API，適合 MVP 快速驗證 aviationstack.com
☔ Real-time Weather APIs
  • Tomorrow.io 超高解析度與分鐘級即時天氣及警報 tomorrow.io/weather-api
  • OpenWeatherMap 即時天氣、分鐘級降雨與極端天氣警報 openweathermap.org/api
  • Apple WeatherKit 精準在地天氣預測與即時警報 developer.apple.com/weatherkit
👥 Crowd Level & POI APIs
  • Google Maps Places API 即時擁擠度 (Current Popularity) 與營業時間 developers.google.com/maps/.../places
  • BestTime.app 景點、商場與餐廳的人潮預測與即時擁擠度 besttime.app
  • PredictHQ 城市級即時事件與大型活動數據 predicthq.com
🚗 Routing & Transit APIs
  • Google Maps Routes API 即時交通時間與大眾運輸替代方案 developers.google.com/maps/.../routes
  • Navitime API 日本在地交通與路線規劃 (Tokyo MVP 首選) api-sdk.navitime.co.jp
